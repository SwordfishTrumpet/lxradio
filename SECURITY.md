# Security Policy

## Supported Versions

Only the latest released version is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest | :x:               |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, report it privately:

- **Email**: security@anomalyco.org (if available) or open a private vulnerability report via GitHub's [Security Advisories](https://github.com/anomalyco/lxradio/security/advisories/new).

Please include:

- A clear description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations

You can expect an acknowledgment within 48 hours and a resolution timeline within 7 days. We'll keep you informed throughout the process and credit you in the release notes unless you prefer to remain anonymous.
